# Magisk DDNS-GO  

## Introduction  

DDNS-GO Magisk Module

## Usage

1.Flash it in Magisk  
2.Reboot yours phone  
3.Connect to Internet and wait for a minute  
4.A website page with URL "[http://127.0.0.1:9876](http://127.0.0.1:9876)" will be opened automatically
